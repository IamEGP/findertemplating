It is necessary to add owl.carousel.css and owl.carousel.min.js plugin files to this page.

<link rel="stylesheet" href="owl.carousel.css"/>
<script type="text/javascript" src="js/owl.carousel.min.js"></script>

To get the plugin working, please use the following script in script.js file.

    $( '.owl-carousel' ).owlCarousel({
        singleItem: true,
        autoPlay: 3000,
        pagination: false,
        paginationSpeed: 1000
    });
with arrow 
    $( '.owl-carousel' ).owlCarousel({
       singleItem: true,
        autoPlay: 3000,
        pagination: true,
        paginationSpeed: 1000,
        navigation: true,
        navigationText: ['<i class="fa fa-angle-left"></i>', '<i class="fa fa-angle-right"></i>'],
        afterInit: htlfndr_move_buttons
    });

For more information on this plugin, please check plugin source code (in js folder).
